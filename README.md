# Hospital-Management-System
📌 Overview

The Hospital Management System (HMS) is a full-stack web application designed to digitalize core hospital operations. It provides two dashboards:

Admin Dashboard – Manage patients, doctors, appointments, admissions, and billing.

Doctor Dashboard – View & complete appointments, view assigned patients.

This project uses:

Frontend: HTML, CSS, TypeScript

Backend: Node.js + Express

Database: MySQL

API: REST APIs built using Express Router

This project is ideal for academic DBMS submissions.
